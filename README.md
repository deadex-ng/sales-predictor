"#sales-prediction"
Web interface for making sales predictions. This is built using streamlit
This is a model that makes predctions about what the sales will be for Rossman Phamarceutical Stores.
Input:

    Store Id,
    DayofWeek,
    Promo,
    SchoolHoliday,
    month,
    year,
    day,
    weekofyear,
    StateHoliday,
    StateHoliday_a,
    StateHoliday_b

Output:
    Predicted sales
